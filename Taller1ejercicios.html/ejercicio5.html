Glosario de Etiquetas HTML

  • <html>: Es la etiqueta raíz de un documento HTML. Define el inicio y el fin del documento.
  • <head>: Contiene metadatos y enlaces externos que no se muestran directamente en la página.
  • <title>: Especifica el título del documento que aparece en la pestaña del navegador.
  • <meta>: Proporciona información de metadatos como charset, descripción y palabras clave para motores de búsqueda.
  • <link>: Vincula recursos externos como hojas de estilo CSS o iconos para la página web.
  • <body>: Contiene todo el contenido visible de la página web, como texto, imágenes, videos, etc.
  • <h1> - <h6>: Representan encabezados en orden de importancia, siendo h1 el más importante.
  • <p>: Define un párrafo de texto en HTML. Se usa para agrupar oraciones y mantener el contenido organizado.
  • <a>: Crea un enlace a otra página, documento o recurso externo. Permite la navegación entre páginas.
  • <img>: Inserta imágenes en una página web. Requiere el atributo src para definir la ruta de la imagen.
