Lenguajes de Programación

  • Python Un lenguaje de programación interpretado, de alto nivel y de propósito general.
  • JavaScript Lenguaje de programación interpretado, principalmente conocido por su uso en páginas web.
  • Java Lenguaje de programación orientado a objetos diseñado para tener pocas dependencias de implementación.
  • C++ Lenguaje de programación compilado, orientado a objetos y de propósito general.
  • C# Lenguaje de programación de Microsoft orientado a objetos y diseñado para la plataforma .NET.
  • PHP Lenguaje de script del lado del servidor originalmente diseñado para la programación web.
  • Ruby Lenguaje de programación interpretado, reflexivo y orientado a objetos.
  • Swift Lenguaje de programación compilado de Apple para macOS, iOS, watchOS, tvOS y Linux.
  • Kotlin Lenguaje de programación estáticamente tipado que funciona en la JVM, Android y el navegador.
  • Go Lenguaje de programación concurrente y compilado inspirado en la sintaxis de C.
