HTML y CSS permiten controlar la alineación de textos para mejorar la presentación de una página web. Alinear textos de manera adecuada puede mejorar la legibilidad. Este es el tercer párrafo, el cual está justificado. La justificación del texto hace que cada línea del párrafo tenga la misma longitud, lo que puede dar un aspecto más formal o elegante. Se usa comúnmente en documentos impresos, como libros o informes. HTML permite justificar fácilmente un párrafo usando CSS.
